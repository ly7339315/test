JS的变量
	var关键字声明一个变量。
		var a = 1;
		a = 2;
		a = 'hello';
		var a = '1';
		var a,b,c;
		var a=1,b=2,c=3;
	变量名命名规则
		1.由字母、数字、下划线、$组成
		2.不能以数字、下划线开始
		3.有特殊含义，采用小驼峰命名法
			sayName
			sayHello
			username
		4.不能使用关键字和保留字
数据类型
	基本数据类型5种
		undefined类型  声明了但是没有初始化的
							显式将undefined值赋值给一个变量的
			就只有一个undefined。
			var a;
			var b = undefined;
		null类型  空，一般用来存放一个空对象的指针。
			就只有一个值null
			var a = null;
		boolean类型
			有两个值true,false
			var a = true;
			var b = false;
		number类型
			数字类型，数值类型
				var n = 1;
				var m = 2.34;
		string类型  字符串类型
			var str = 'hello';
			var s = "hello";
	引用数据类型3种-1种
		Object类型 对象
			var obj = {};
			var o = {name:'zhangsan'};
		Array类型  数组
			var arr = [1,2,3];
		Function类型  函数
			function test(){}
			var test = function(){}
